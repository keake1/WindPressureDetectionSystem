STC8H系列芯片使用注意事项：
除P3.0/P3.1外，其余所有I/O口上电后状态均为高阻输入状态，用户在使用I/O口时必须先设置I/O口模式
////////////////////////////////////////////////////////////////////////////////////////////
////////////////////////////////////////////////////////////////////////////////////////////
* I/O端口模式说明:
	STC8系统单片的所有IO都可以通过软件为以下4种工作模式:
	工作模式			配置方法
	准双向口/弱上拉		P[0:7]M1 = 0, P[0:7]M0 = 0
	推挽输出/强上拉		P[0:7]M1 = 0, P[0:7]M0 = 1
	输入模式/高阻		P[0:7]M1 = 1, P[0:7]M0 = 0
	开漏输出			P[0:7]M1 = 1, P[0:7]M0 = 1
	
* 设置端口模式

例如：设置P0口为准双向口
	P0M1=0x00;
	P0M0=0x00;    
	
需要注意：
	P10 P14复位后为强推挽输出、强上拉，防止不当操作导致端口损坏，初始化为准双向口，将P10 P14设置为准双向口
	P1M1 &=~( (1<<0) | (1<<4) );  
	P1M0 &=~( (1<<0) | (1<<4) );    	
分析：
	1<<0等价于0x01 即 0000 0001
	1<<1等价于0x02 即 0000 0010
	1<<2等价于0x04 即 0000 0100
	1<<3等价于0x08 即 0000 1000
	以此类推1<<n 即第n位为1，其余位是0
1）x |=(1<<n)  即对x执行按位取或，即x中的第n位置为1，不改变其他位状态  
2）y &=~(1<<n) 即将1<<n按位取反，然后对y按位取与，即y中的第n位置为0，不改变其他位状态
